Ciencia de datos con Python

Comprende los conceptos básicos de programación en Python y aplica los principios de la programación orientada a objetos (POO) para resolver problemas simples.

Desarrolla aplicaciones utilizando técnicas de machine learning como regresión, clasificación y clustering, integrando modelos para la toma de decisiones basada en datos.

Duración: 32 horas Académicas
